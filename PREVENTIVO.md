## DESCRIZIONE DEL PROGETTO

Sviluppo completo di un menù digitale interattivo per ristorante con sistema di gestione contenuti (CMS) integrato, ottimizzato per dispositivi mobile e desktop.

---

## SPECIFICHE TECNICHE REALIZZATE

### 1. ARCHITETTURA E INFRASTRUTTURA
- **Hosting:** Netlify (gratuito, incluso SSL/HTTPS)
- **Repository:** GitHub (gratuito, con versioning completo)
- **CMS:** Netlify CMS integrato con autenticazione
- **Tecnologie:** HTML5, CSS3, JavaScript (Vanilla)
- **Build System:** Automatizzato con script Node.js

### 2. SISTEMA DI GESTIONE CONTENUTI (CMS)

#### 2.1 Pannello Amministrativo
- Accesso protetto con autenticazione Netlify Identity
- Interfaccia intuitiva per gestione autonoma
- Sistema "Invite Only" per sicurezza
- Git Gateway per sincronizzazione automatica

#### 2.2 Categorie Gestibili
**Birre (4 sezioni):**
- Birre artigianali alla spina a rotazione
- Birre alla spina
- Birre speciali in bottiglia
- Frigo Birre

**Altre Bevande (7 categorie):**
- Cocktails
- Analcolici
- Bibite
- Caffetteria
- Bollicine
- Bianchi fermi
- Vini rossi

#### 2.3 Campi Personalizzabili per Prodotto
- Nome prodotto
- Immagine grande (opzionale)
- Logo piccolo (opzionale)
- Descrizione breve (max 500 caratteri)
- Descrizione dettagliata (max 2000 caratteri)
- Prezzo
- Formato (es: Boccale 0,5L, Calice, Bottiglia)
- Gradazione alcolica
- Categoria/Tipo
- Tag speciali (Novità, Senza Glutine, Specialità, Biologico)
- Allergeni (Glutine, Lattosio, Solfiti, Frutta a Guscio, Sedano, Uova)
- Disponibilità (Sì/No)
- Ordinamento personalizzato

### 3. INTERFACCIA UTENTE

#### 3.1 Design e Branding
- Logo Arconti31 integrato nell'header
- Palette colori personalizzata
- Design moderno e professionale
- Animazioni fluide e transizioni

#### 3.2 Navigazione
- **Vista Home:** Categorie cliccabili con contatore prodotti
- **Vista Dettaglio:** Prodotti della categoria selezionata
- **Vista Compatta:** Modalità lista condensata (solo nome e prezzo)
- Breadcrumb navigation per tornare alla home
- Supporto tasto ESC per navigazione rapida

#### 3.3 Visualizzazione Prodotti
- **Card Standard:** Immagine grande, descrizione completa, tag, allergeni
- **Card Logo:** Solo logo piccolo accanto al nome (layout compatto)
- **Popup Dettaglio:** Click su prodotto per informazioni complete
- Badge colorati per tag speciali con icone emoji
- Badge allergeni con icona warning
- Indicatore disponibilità con stato visivo

#### 3.4 Funzionalità Avanzate
- Filtri per categoria birre (Chiara, Scura, Rossa, ecc.)
- Toggle vista compatta/completa
- Popup modale con dettagli estesi
- Chiusura popup con click esterno o ESC
- Lazy loading immagini per performance

### 4. RESPONSIVE DESIGN
- **Mobile First:** Ottimizzato per smartphone
- **Tablet:** Layout adattivo
- **Desktop:** Griglia multi-colonna
- Touch-friendly per dispositivi mobili
- Font e spaziature ottimizzate per ogni dispositivo

### 5. FOOTER E INFORMAZIONI
- Indirizzo cliccabile: Via Vittorio Arconti, 31, 21013 Gallarate VA
- Link diretto a Google Maps per indicazioni
- Copyright e branding
- Design elegante con hover effects

### 6. PERFORMANCE E OTTIMIZZAZIONE
- Caricamento veloce (< 2 secondi)
- Immagini ottimizzate con lazy loading
- CSS e JavaScript minificati
- Cache browser ottimizzata
- SEO-friendly con meta tags

### 7. SISTEMA DI BUILD AUTOMATICO
- Script di generazione JSON da markdown
- Parsing YAML per dati strutturati
- Raggruppamento automatico per sezioni
- Deploy automatico su push GitHub
- Rebuild automatico su modifiche CMS

### 8. DOCUMENTAZIONE FORNITA
- **DEPLOY_GUIDE.md:** Guida completa al deploy (step-by-step)
- **GUIDA_RISTORATORE.md:** Manuale d'uso del CMS
- **ARCHITETTURA.md:** Documentazione tecnica del sistema
- **README.md:** Panoramica del progetto
